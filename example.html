General features

Download and install it with:
npm i
It will install
node-sass
gh-pages(as dev dependency)
package.json contains the following scripts:
"scripts": {
"sass": "node-sass --source-map dist/css/main.css.map --output-style compressed -w scss -o dist/css -- recursive ",
"publish": "gh-pages -d dist"
}
Therefore for sass:
npm run sass
To publish it be sure the git repo is already synced and then just run:
npm run publish

How buttons work

Under "components/_buttons.scss"
The function takes those arguments:

$color | $color-hover | $border-color | $text-norm | $text-hover | $type | $border | $size

&-blue {
@include button( $light-blue, $light-blue-hover, $light-blue, $light-blue-text-norm, $light-blue-text-hover, solid, 5px, normal );
}

&-blue-out {
@include button( $light-blue, $light-blue-hover, $light-blue, $light-blue-text-norm, $light-blue-text-hover, outlined, 20px, small );
}

How Forms and Input Work

Simple Input for a form with .input-text-email

                
<input
class="input-text-email"
type="email"
placeholder="example@email.com"
/>

Simple Input for an email with direct subscription with .newsletter 

                
<div class="newsletter">
   <input
   class="input-text-email"
   type="text"
   placeholder="Email"
   name="email"/>
   <button>Submit</button>
</div>

Multiple selection .select-css

                      
<select class="select-css" name="cars" id="cars">
  <option value="" disabled selected>Please Select</option>
  <option value="volvo">Volvo</option>
  <option value="saab">Saab</option>
  <option value="mercedes">Mercedes</option>
  <option value="audi">Audi</option>
</select>

Simple Input text area with .input-text-area

                      
<textarea
  class="input-text-area"
  rows="5"
  cols="60"
  name="description">
  Enter details here...
</textarea>

How Margins and Paddings work

Under "variables/_aligns.scss" you can modify the functions and add or remove values
For margins: .mb-1~4 .mt-1~4 .mr-1~4 .ml-1~4 .mv-1~4 .mh-1~4
For paddings: .pb-1~4 .pt-1~4 .pr-1~4 .pl-1~4 .pv-1~4 .ph-1~4
This is an example that gives a padding horr + vert of 1rem 
                
               <div class="col-12 ph-1 pv-1">
                <h4>This is an example</h4>
              </div>
